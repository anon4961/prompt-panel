# Prompt Panel

SillyTavern의 프리셋, 월드인포, 캐릭터카드를 한 패널에서 열람·번역·검색·내보내기 할 수 있는 확장입니다.

## 주요 기능

### 데이터 로드
현재 사용자의 SillyTavern에 등록된 **프리셋, 월드인포, 캐릭터카드**를 패널에서 직접 로드해서 보여줍니다.

### 번역
- 토글/엔트리/필드 단위 번역, 개별 재번역
- 번역 캐시는 브라우저 IndexedDB에 저장 (SillyTavern 설정 파일에 저장하지 않음)
- 지원 프로바이더: OpenAI, Claude, Google AI Studio (MakerSuite), Vertex AI, OpenRouter, DeepSeek, Mistral, Groq, Cohere, xAI
- 프로바이더별 샘플링 파라미터 개별 설정

### 검색
원문과 번역문 양쪽 모두에서 키워드 검색이 가능합니다. 일치하는 부분은 패널에서 하이라이트로 표시됩니다.

### 토큰 수 계산
로드된 항목 전체 토큰 수, 선택한 토글의 합계 토큰 수가 실시간으로 표시됩니다.

### 복사 및 내보내기
각 탭(프리셋/월드인포/캐릭터) 상단 우측에 3개의 아이콘 버튼이 있습니다.

- **복사**: 클립보드로 복사
- **TXT 내보내기**: 텍스트 파일로 저장
- **JSON 내보내기**: SillyTavern 호환 JSON 파일로 저장

#### 복사 / TXT 내보내기
토글을 일부만 선택해서 그 부분만 복사·내보낼 수 있습니다. 실행 시 **원문만 / 번역문만 / 둘 다** 중 하나를 선택할 수 있고, 번역이 없는 항목은 번역문 모드에서 `(번역없음)`으로 표시됩니다.

#### JSON 내보내기
- **프리셋**은 토글 선택 여부와 관계없이 **항상 전체**를 내보냅니다 (구조가 깨지지 않도록 보장). 프리셋에 동봉된 **정규식과 기타 확장 설정도 함께 내보내집니다**.
- **월드인포**는 토글을 일부만 선택했다면 선택된 엔트리만 내보냅니다.
- **캐릭터카드**는 필드를 일부만 선택했다면 선택된 필드의 번역만 적용해서 내보냅니다. **임베디드 월드북(`character_book`)이 카드 안에 있다면 함께 내보낼지 묻는 창**이 뜨고, 포함을 선택하면 원문 그대로 동봉됩니다.
- 토글 제목도 번역되며, JSON 내보내기 시 번역된 제목은 토글의 이름 필드(프리셋의 `name`, 월드인포의 `comment`)에 들어갑니다.

#### 파일명
`{소스이름}_{번역언어}.{확장자}` 형식으로 자동 생성되어 브라우저의 기본 다운로드 폴더에 저장됩니다.

### UI
- **플로팅 아이콘**: 옵션으로 활성화 시, 화면에 상시 떠 있는 드래그 가능한 아이콘으로 어디서든 패널을 열 수 있습니다
- 프리셋/월드인포/캐릭터 탭
- 패널 헤더의 새로고침 버튼: ST에 새로 추가/삭제된 항목을 즉시 패널에 반영 (번역 캐시는 보존)
- 원문/번역문 글자 크기 조절
- 테마 6종: Dark, Light, Pink, Mint, Orange, Blue

### 지원 언어
한국어, 영어, 일본어, 중국어 간체, 중국어 번체, 폴란드어.

## 설치

다음 두 가지 방법 중 하나를 사용하세요.

1. SillyTavern의 확장 매니저에서 URL로 설치  
2. 본 저장소의 zip 파일을 직접 다운로드하여 SillyTavern의 `data/default-user/extensions/` 폴더 안에 압축 해제

## 참고 사항

- 모든 내보내기는 원본 데이터를 깊은 복제(deep clone)한 뒤 복제본만 수정하므로, **원본 프리셋/월드인포/캐릭터카드는 절대 변경되지 않습니다.**
- 캐릭터카드 JSON 내보내기는 V3 spec에 따라 `data.*` 경로를 정본으로 사용하며, V1 호환용 최상위 중복 필드는 비워서 파일 크기를 최소화합니다. SillyTavern은 import 시 `data.*`를 우선 사용하므로 호환성에 영향이 없습니다.
